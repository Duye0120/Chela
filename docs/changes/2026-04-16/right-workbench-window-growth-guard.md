# 右侧工作区窗口重复扩宽修复

**时间**: 2026-04-16 13:45

## 改了什么

1. 给右侧工作区开关流程增加了切换中的互斥保护，避免连续点击时重复执行窗口扩宽或回收。
2. 打开 Diff 时优先复用首次记录的原始窗口 bounds，避免把已经扩宽后的当前宽度继续当成新基线。

## 为什么改

- 连续点击 Diff 开关时，窗口扩宽逻辑会重复叠加，表现为“越点越宽”。
- 这类窗口级布局动作需要单次串行执行，且基线必须固定在展开前的原始尺寸。

## 涉及文件

- `src/renderer/src/App.tsx`

## 结果

- 重复点击 Diff 开关不会再把窗口宽度一层层叠上去。
