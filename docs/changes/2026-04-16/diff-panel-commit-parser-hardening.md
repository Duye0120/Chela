# Diff Panel 提交信息解析加固

**时间**: 2026-04-16 12:06

## 改了什么

1. 加固 `src/main/worker-service.ts` 里的 commit 结果解析逻辑。
2. 兼容工具模型常见输出格式：JSON、`标题/描述` 标签式、多一层 markdown heading 或列表前缀。
3. 当标题仍然无法提取时，把模型原始返回的预览片段带进错误信息，方便直接定位具体输出形态。

## 为什么改

- diff-panel 的星按钮已经能触发工具模型生成，当前错误发生在“模型输出转成 title / description”的解析阶段。
- 部分工具模型会返回 `标题：` 换行后再给正文，或直接返回 JSON；旧解析只认“第一行直接就是标题”。
- 需要把失败从黑盒变成可见信息，减少用户和开发者排查成本。

## 涉及文件

- `src/main/worker-service.ts`

## 结果

- commit 生成对真实模型返回更宽容。
- 再次失败时，diff-panel 错误框会带出原始返回预览，更容易看出模型到底回了什么。
